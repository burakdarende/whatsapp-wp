# Multi-Branch WhatsApp Sticky Button

WordPress için sabit WhatsApp butonu: birden fazla hat tanımlanabilir; tek hat doğrudan, birden fazlaysa menüden seçim.

## Kurulum

1. `multi-branch-whatsapp-sticky-button` klasörünü `wp-content/plugins/` içine kopyalayın.
2. Yönetimden eklentiyi etkinleştirin.
3. **Ayarlar → WhatsApp Hatları** üzerinden numaraları kaydedin (ülke kodu ile).

**Gereksinim:** WordPress 5.8+, PHP 7.4+
